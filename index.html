In this problem, we will model a quantum dot (e.g., a CdSe nanoparticle) as a particle (electron) in a cubic box with side d. Consider what happens if we apply some force to one of the facets so that the cube gets distorted, but its volume remains the same (V = d^3). a = d(1 + epsilon), b = d(1 - epsilon), c = d, with abc = V = d^3 = constant. Express the deformation parameter using the volume constraint and apply a Taylor series expansion assuming epsilon << 1.
